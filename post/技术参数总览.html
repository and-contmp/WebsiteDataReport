      WEB框架&代码规范:

      Hexo 3.0 开源静态生成系统

      源代码库地址:hexo for github

      安装方式

      npm install hexo-cli -g


      开发环境支持:

      NODEJS
      官网下载最新版64位安装程序,使用内置的npm包管理器

      npm install -g npm

      更新后只用npm安装Hexo库及后续插件

      Hexo插件库

      Hexo3.0后,将以下插件独立出去,需要单独安装:

      npm install hexo-renderer-ejs --save
npm install hexo-renderer-stylus --save
npm install hexo-renderer-marked --save


      EJS模板技术

      直接下载源码集成到系统开发,技术文档地址:google代码库



      编程语言:HEML5+CSS3+JAVASCRIPT+NODEJS

      符合Html5和css3代码规范
      Javascript代码遵循CommonJS规范标准
      Nodejs是一个服务器端的Javascript标准实现,采用Chrome V8 引擎的Javascript运行环境
      兼容CommonJS和ES6编码规范
      CommonJS标准查询
      ES6

      数据可视化类库:

      Echarts3
      官网下载 Echarts.min.js到本地使用
      技术文档地址:Echarts3 API

      使用的开发工具:

      ###Microsoft Visual Studio Code for Windowsx64
      版本号:Version 1.12


      ​ VScode详情

      Cygwin64 Terminal

      版本号:2.7.6.0
      因为win7系统没有SSH端口,需要Cygwin虚拟UNIXshell开启SSH端口,为部署和上传做准备
      Cygwin详情

      GIT

      版本号:2.13.0.1
      用来部署网站和上传数据到GITHUB

      ​ GIT详情

      GitHub

      GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub
      数据分析报表系统部署在github上,为报表体统提供免费空间

      MarkDown

      所有内容页使用markdown格式编写存档,部署时再生成静态THML文件上传至服务器
      MarkDown详情

      作用和意义

      1. 提供统一的,高效率,清晰一致的分析数据展示接口
      2. 目前只涉及网站数据,之后可根据具体业务需求增加数据领域和功能需求
      3. 网站技术问题记录和说明
      4. 存档留底
